About priceTracker

priceTracker is a powerful, user-friendly macOS application designed to keep you informed about the latest stock and cryptocurrency prices in real time. Completely free to use, it updates every minute to ensure you always have the most accurate and up-to-date market information at your fingertips. Whether you're a seasoned trader or just starting your investment journey, priceTracker is a reliable companion for staying on top of the markets effortlessly. This program will be indefinitely hosted on github due to Apple's lack of privacy for small developers who utilize their app store.

Functionality

priceTracker lets you effortlessly track any of the top 50 cryptocurrencies and stock tickers without the need for API keys or complex setups.
Simply open the app, select the widget, and enter your tickers using : as a delimiterâ€”for example:

LTC:BTC:GOOG:NDAQ

You can add as many or as few tickers as you like.
priceTracker will automatically update the data every minute using reliable sources like Yahoo Finance and CoinGecko.

What To Expect

priceTracker is a lightweight and efficient program designed to deliver all the information you need while running so efficiently, your computer doesnt even know it's there.
Optimized to minimize API usage, it remains free to use indefinitely, filling the gap for a publicly available tool that tracks both crypto and stock tickers in one simple application.
Should you encounter API throttling from frequent ticker changes, just wait a minute for the system to reset.
Tickers update every 60 seconds.
